    Welcome to Project Pluto's home page.

    Pluto is a visual programming environment which aims to provide aspiring students the ability to learn to code with a low floor, and a high ceiling. A main differentiator between Pluto and other similiar applications (such as Scratch) is the flexibility it provides to create unique culturally situated tools to enhance STEM education.
